## What is this?

AIELTS is an Project that leverage ChatGPT, it mainly provide the following service:

1. Mock IELTS Writing question library to user for training their writing skills.
2. Provide estimate Grade, Comments and Grammer check by using the ChatGPT.
3. Using stripe for token selling, and user can use the token for different AIELTS

## Difficulty & Solution

1. Ensuring accurate and effective grading of user writing, as this is a subjective process with many variables.

   - By using ChatGPT model it can provide great sample writing and also basic grammer check.
   - We can use the chatgpt api to generate response for the writing, but it still need to play time to modify the format of response to ensure it can be display in the Web

2. Facilitating secure and smooth transactions for the token system might pose certain technical challenges.

- Integration with Stripe, a leading online payment processing platform, ensures secure and reliable transactions. Stripe's APIs allow for easy implementation and provide extensive support for developing the token system.
